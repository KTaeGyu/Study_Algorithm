# 재귀 호출
: 자기 자신을 호출하여 순환 수행되는 것
- 종료 조건을 명확히, 반드시 포함
- 반복되는 호출이 종료 조건으로 향하도록 해야함

## 예시1: 팩토리얼
  - $n! = n * (n - 1)!$
  - $f(n) = n * f(n - 1)$
```python
def factorial(n):
    if n == 1:
        return 1
    else:
        return n * factorial(n - 1)
```
